TaDa Application

Web application that is a clone of Trello and Google Calendar. On the left, you can create notes. On the right, you can add events to a calendar.

Sign-In

To sign-in, click the Sign-In button that will allow you to access the application with Google account. Upon login, you will be greeted with the TaDa page. From here, you will be able to create notes and calendar events. When you re-enter the application after the log-out, the saved notes and events from the web server database will populate.

Note

When you create the note, you can write the title and description of the note. Additionally, you can drag the note and post it on the other location. Moreover, you are able to delete the note by clicking the upper-right button.

Calendar

When you add the event to calendar, you can write the title, start time, and end time of the event. Once you click the start and end time, the modal will allow you to choose the date, hour, and minute. After saving, the event will appear in the calendar, which is clickable to edit or delete.

Weekly View

For calendar, the weekly view is provided along with the monthly view. On the upper-right corner of the calendar, clicking each button will switch to the corresponding view.

Bug Report

1) To report bug, you should click the link on the right, which will take you to the GitHub issues page for TaDa web application.
2) Click the New Issue button and insert the title according to the format. Bug Report: (content).
3) In the description, write the details about the bug that you found.
4) On the right, label the issue by clicking the label and bug.
5) To finish the submission, click the Submit New Issue button.
